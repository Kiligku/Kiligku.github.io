# HEXO上传博客常用指令
<!--more-->
### hexo n title
新建指定名字的博客
### hexo clean
清除缓存文件和已经生成的静态文件
### hexo g
生成网站的静态文件到public文件夹中
### hexo d
生成网站的静态文件并部署到仓库中
### hexo s
启动本地服务器，即输入网址 http://localhost:4000/ 就可以在本地预览网页

### 常用组合
- hexo n + hexo clean + hexo g + hexo s:生成本地可以查看的网页
- hexo n + hexo clean + hexo g + hexo d:生成远端可以查看的网页

### 在vscode中上传博客
终端-新建终端-右下加号-选择git bash-输入相应指令
